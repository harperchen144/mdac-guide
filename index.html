Registration
Personal Information 个人信息
所有内容必须与护照完全一致，建议打开护照对照填写
* Name : 键盘输入
填护照上的英文全名，大小写无所谓，拼写要和护照一致
✍️ 建议填写：护照姓名页上的英文名，如 ZHANG SAN 或 TAN WEI MING
* Passport No. : 键盘输入
✍️ 填护照号码，如 E12345678
* Nationality / Citizenship : 下拉选择
下拉列表里找你的国籍
中国选 CHINA，新加坡选 SINGAPORE
* Date of Birth : 下拉选择
选出生日期，注意是 日/月/年
1990年3月15日 ➔ 15/03/1990
* Place of Birth : 下拉选择
选出生国家，和国籍通常一样
* Sex : 下拉选择
男选 Male，女选 Female
* Date of Passport Expiry : 下拉选择
选护照有效期
2030年9月20日 ➔ 20/09/2030
* Email Address : 键盘输入
✍️ 填常用邮箱，如 zhangsan@email.com，提交后会收到确认邮件
* Confirm Email Address : 键盘输入
✍️ 重复填一遍上面的邮箱，必须完全一样
* Country / Region Code : 下拉选择
选你目前居住的国家
住新加坡选 SINGAPORE，住中国选 CHINA
* Mobile No. : 键盘输入
🚨 前面必须加国家区号！
✍️ 新加坡 65-9xxxxxxx
中国大陆 86-138xxxxxxxx
香港 852-9xxxxxxx / 台湾 886-9xxxxxxxx
Registration
Traveling Information 行程信息
⚠️ 到达日期必须在提交后 3 天内（含当天），提前太多会填不了！
* Date of Arrival : 下拉选择
选实际入境日期
2026年8月2日 ➔ 02/08/2026
* Mode of Travel : 下拉选择
陆路过关选 Land
飞机选 Air / 坐船选 Sea
* Date of Departure : 下拉选择
选离开马来西亚的日期
2026年8月6日 ➔ 06/08/2026
* Last Port of Embarkation before Malaysia : 下拉选择
进入马来西亚前你实际所在的国家
从新加坡过来选 Singapore
* Flight / Vessel / Transportation No. : 键盘输入
✍️ 陆路没有航班号，直接打 NA
* Accommodation of Stay : 下拉选择
住酒店选 Hotel，住朋友/亲戚家选 Others
Registration
Address (In Malaysia) 马来西亚住址
⚠️ 只能填英文字母和数字 (alphanumeric only)，不能输中文！
* Address : 键盘输入
✍️ 照抄上面两行地址即可
* State : 下拉选择
选 Johor
* City : 下拉选择
先选完 State 才出现 City 列表
选 Bandar Penawar
* Postcode : 键盘输入
✍️ 填 81930

Drag To Verify 拖动滑块验证
➔
Slide the Puzzle ➔
向右拖动滑块完成拼图验证（类似验证码）
SUBMIT
全部检查无误后点击提交

💡 提交后注意事项

  1. 提交成功后你的邮箱会收到一封确认邮件
  2. 一定要截图保存确认页，记下注册号码（Registration No.）
  3. 入境马来西亚时海关可能会抽查，手机里存好截图备用
  4. 万一填错了不用慌，重新注册一次就行，系统以最后一次提交为准
